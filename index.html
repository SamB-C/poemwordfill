Guide

See below for a simple 5-step guide on how to use samsonbc.com to revise poetry!

1. Select Anthology

Click on the name of the current anthology.

Use the dropdown menu to select the anthology you wish to revise.

2. Select Poem

Click on the title of the current poem.

Use the dropdown menu to select the poem you wish to revise.

3. Choose “Words” or “Quotes”

Use the radio buttons to select either “Words” or “Quotes”.

Words

This setting removes words at random from the selected poem.

Quotes

This setting removes quotes at random from the poem, from a specially curated list of quotes deemed particularly useful to memorise.

4. Select Difficulty

Use the slider to select the percentage of words or quotes that you wish to remove.

5. Fill in the blanks!

Each of the removed words appears as a blank. Try to fill them in. If you get a letter wrong 3 times, it gives you the answer.

Viewing Notes (Computer Only)

Hover your mouse over different phrases in the poem to reveal curated notes about them.
